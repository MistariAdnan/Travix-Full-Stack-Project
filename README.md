# 🚀 Travix – Full Stack Property Listing Platform

Travix is a full-stack web application that allows users to explore, create, and manage property/hotel listings. It includes authentication, image uploads, booking functionality, and a responsive UI.

---

## 🌐 Live Demo
👉 https://travix-stay.onrender.com/listings

---

## 📌 Features

- 🔐 User Authentication (Signup/Login/Logout using Passport.js)
- 🏠 Create, Edit, Delete Listings (CRUD operations)
- 📸 Image Upload with Cloudinary
- 🔍 Search & Filter Listings
- ⭐ Review & Rating System
- 📅 Booking System (Reserve listings)
- 💬 Flash Messages for user feedback
- 📱 Responsive UI (Bootstrap)

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- Bootstrap
- EJS (Templating)

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB Atlas
- Mongoose

**Authentication:**
- Passport.js
- express-session

**Other Tools:**
- Cloudinary (Image Storage)
- Multer (File Uploads)
- Render (Deployment)

---

## ⚙️ Installation & Setup

1. Clone the repository
git clone https://github.com/YOUR_USERNAME/Travix.git

2.Install dependencies
npm install

3.Create .env file and add:
DB_URL=your_mongodb_url
SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=xxx
CLOUDINARY_KEY=xxx
CLOUDINARY_SECRET=xxx

4.Run the app
node app.js
Open in browser:
http://localhost:8081
